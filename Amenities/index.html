Amenities

Essentails but better

Features

Join Leave Messages

Displays a custom join leave message when a player joins or leaves the server. This can be disabled in the config.

Custom Chat Format

Changes chat messages to look cleaner, neater and display player ranks (Requires Vault)

Spawn System

Custom spawn system allows users to teleport back to spawn at any time. Also contains settings modifying teleport time and allowing pvp during teleport And More.

Vault Support

Allows vault prefixs, suffixs, and nicknames to be used in custom chat messages.

Shortcuts

Allows server operators to change their gamemode quickly without needing to type the full "/gamemode Creative". Also contains teleport shortcuts to teleport to a player or teleport a player to you.

Auto Complete

All commands have auto complete built in, meaning you don't need to remember the name of a command function.
